Renders text with support for styling via HTML tags. The rendering can vary across browsers and platforms.

## Usage

```svelte live
<script>
  import { HTMLText } from 'svelte-pixi'
</script>

<HTMLText
  x={360}
  y={200}
  text="<strong>Hello</strong> <em>World</em>"
  anchor={0.5}
  style={{
    fill: 'white',
  }}
/>
```
